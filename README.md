# Automated Payroll Dashboard - 50 Employees

## What I Did:
- **Large Dataset Management:** Built complete payroll dashboard for 50 employees across different departments
- **Tax & Salary Logic:** Automated formulas to calculate Gross Salary, Taxes, and Net Salary
- **VLOOKUP Integration:** Smart search feature to instantly fetch any employee's details by ID
- **Allowance Automation:** Logical formulas to calculate House Rent, Medical, and Utility allowances based on Basic Salary

## Tools Used:
Microsoft Excel (VLOOKUP, Financial Formulas, Tax Logic, Data Management)
